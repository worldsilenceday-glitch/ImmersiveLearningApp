## Overview
The **Immersive Learning App** is an interactive 3D educational application for kids aged 8-14.  
It allows learners to explore planets, biology models, and other subjects through immersive experiences, quizzes, and interactive controls.

## Features
- Interactive 3D models with rotation and zoom
- Educational quizzes with scoring and feedback
- Progress tracking and save/load functionality
- Modular UI system with topic selection menu
- Fun, colorful, and child-friendly design

## Build Instructions

### Windows Executable
1. Open the project in Unity Hub
2. Go to **File > Build Settings**
3. Select **PC, Mac & Linux Standalone**
4. Choose **Windows** as the target platform
5. Click **Build and Run** and choose a location

### WebGL Version
1. Open the project in Unity Hub
2. Go to **File > Build Settings**
3. Select **WebGL** as the target platform
4. Click **Build and Run**
5. Upload the output folder to a web server
