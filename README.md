
## Build & Run Instructions

### Windows Executable
1. Open the project in Unity Hub
2. Go to **File → Build Settings**
3. Select **PC, Mac & Linux Standalone**
4. Choose **Windows** as target platform
5. Click **Build and Run** and select a folder to save the executable

### WebGL Version
1. Open the project in Unity Hub
2. Go to **File → Build Settings**
3. Select **WebGL** as target platform
4. Click **Build and Run**
5. Upload the contents of the folder to a web server

## System Requirements
- Unity 2021.3 LTS or higher recommended
- Windows 10/11, macOS 10.14+, or Linux equivalent
- OpenGL 3.3 or higher graphics card
- Optional: Android/iOS modules for mobile builds
